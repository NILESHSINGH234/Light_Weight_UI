Image

Image components are used in visual representation which can be screen responsive or can be in circle shape.

Responsive Image

The responsive image expands to cover the parent container. Add img-responsive class and resize the browser window to see the responsiveness.

Round Image

Round image has a circle shape which can be used as profile pic avatar etc. It has default dimension of 160px X 160px.

Image with Rounded Corners

This image has a slight border-radius at the corners.
