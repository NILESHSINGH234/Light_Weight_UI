Button

Buttons are used as CALL TO ACTION, available in different sizes and states.

Basic Buttons

The basic button has 7 different states - primary, secondary, success, danger, warning, light & dark.

Outline Buttons

The outline button has no background color. You can use it as secondary call to action.

Link Buttons

The link button has anchor tag which enables link action.

Buttons with Icon

These buttons have icon which enhance the overall experience.

Floating Action Buttons

Button Sizes

Buttons are available in 2 different sizes - btn-lg & btn-sm

Utilities

Utility classes will provide more customisation - .btn-disabled class which makes the button disabled and .btn-shadow which provides a little elevation to the button.
